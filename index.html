Beberapa Projek yang di buat

Desain tampilan front-end aplikasi mobile Android ini dibuat menggunakan Flutter. Flutter digunakan untuk menciptakan antarmuka pengguna yang responsif dan menarik, dengan pengalaman pengguna yang mulus. Aplikasi ini menampilkan gambar dengan tombol navigasi untuk mengganti gambar, memberikan pengalaman interaktif bagi pengguna. Selain itu, aplikasi ini juga dilengkapi dengan fitur e-commerce yang memungkinkan pengguna untuk menjelajah produk, melakukan pembelian, dan menikmati pengalaman berbelanja yang mudah dan nyaman.

Website ini dibangun menggunakan Python sebagai bahasa pemrograman utama dan SQLite sebagai basis data. Dalam proses pengembangan, Python digunakan untuk mengimplementasikan algoritma analisis sentimen, sedangkan SQLite digunakan untuk menyimpan data ulasan yang dianalisis. Website ini dirancang untuk memberikan hasil analisis yang cepat dan akurat.
